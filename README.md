ublas_performance_sandbox
=========================

A sandbox repo for ublas performance experiments.
To download:
-----------
https://github.com/nasosi/ublas_performance_sandbox.git

To compile:
-----------
qmake

make

or:

g++ -DNDEBUG -O3 -std=c++0x main.cpp -o benchmarks

To run:
---------
./benchmarks
